# Spinning-Cube

Quick project to be more familair with Rust.
The objective was to implement in Rust a spinning cube displayed on a terminal in ascii.
This project is basically a translation of saatvikrao's [work](https://github.com/saatvikrao/Spinning-Cube) from c to Rust.

## Files
Everything is in `src/main.rs`.

## How to run
While in the repository, run the following command to compile the project and run it:
```console
cargo run
```
